## smart-led

> 智能点阵屏

功能

- 软件模拟SPI
- 16*16汉字显示
- 多种显示模式
- 蓝牙控制

硬件

- Arduino Nano
- 16*16点阵屏
- HC-05

连线说明

| 16*16点阵屏 | Arduino Nano | 备注     |
| ----------- | ------------ | -------- |
| GND         | GND          | 接地     |
| VCC         | 5V           | 电源     |
| OE          | 6            | 138使能  |
| R           | 8            | 595输入  |
| SCK         | 9            | 595时钟  |
| LATCH       | 10           | 595输出  |
| A           | 2            | 138输入1 |
| B           | 3            | 138输入2 |
| C           | 4            | 138输入3 |
| D           | 5            | 138输入4 |
